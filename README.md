## Installation Guide
### Prerequisities
System dependencies:
```
$ ruby -v
  2.6.1
```

#### Local setup
##### Clone project
```
$ git clone https://github.com/madcido/purplebook.git
$ cd ./purplebook
```
##### Install dependencies
```
$ bundle install
```
##### Build project
```
$ rails s
```
